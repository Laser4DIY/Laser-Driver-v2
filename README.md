# LASER4DIY Laser Diode Driver

Laser diode driver to be used for the Laser4DIY project, see http://www.laser4diy.org/

**Note: This project is work in progress!**

## Design Goals

Continous mode: up to 80A (at 2V)
Pulse mode: up to 80A (at 2V), pulses <50µsec possible
